Methods

inherits(parent)

Allows class inheritance. The calling class will inherit the specified parent class's properties. This also adds a new property to the child class called `parent`. This lets the child class access the parent's original properties; i.e., like the super keyword in Java.
Parameters:
Name Type Description
parent function The class that the calling class will inherit from.
Source:
  • util/Class.js, line 17
Example
function ParentClass() {
    this.thing = "hello";
}
ChildClass.inherits(ParentClass);
function ChildClass() {
    ChildClass.parent.constructor.apply(this, arguments); // Call parent constructor
}
console.log((new ChildClass()).thing); // outputs "hello"
